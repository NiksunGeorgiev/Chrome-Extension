This is a Chrome Extension with the following futures :

1.Open a page (by URL)
2.Keeps the page open
3.Reloads the page (with interval)
4.It has a popup page for user config
5.Starts and stops the reload interval
6.Shows Last refresh / Next refresh
7.After each page reload
a)Fetch a random image from an API
b)Insert it on the page, cantered, in a dialog box

Instructions to run the project :

1.clone the repository to your local repository 
2.open the terminal  add npm install - > this will install all dependencies and dev/dependencies 
3.add npm install - > this will install all dependencies and dev/dependencies 
4.add npx weback to build the project and loaded with Babel.


How to add the extension in Chrome : 

Navigating to extensions in Chrome
After choosing Extensions, it redirects to the extensions page in Chrome. Make sure to enable the Developer mode here.
Once that's done, you need to click the Load unpacked button which will allow us to load our project in the Chrome extension store.
